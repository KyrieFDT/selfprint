自助打印系统 · 使用说明

适用于照相馆 / 文印店 | 版本 v1.0

一、这个系统是干什么的

顾客来店里打印文件，不用再排队等店员操作电脑。

顾客用自己手机扫码 → 上传文件 → 预览排版 → 选好设置 → 到前台付款 → 打印机自动打印。

店员只需要确认收款，系统自动算钱、排队、打印一条龙。

二、第一天怎么用（5 分钟）

你需要准备

  • 一台 Windows 电脑（Win10 或 Win11）
  • 电脑已经连着打印机（平时能正常打印就行）
  • 店里有 WiFi（顾客手机要连，和电脑同一个 WiFi）

操作步骤

1 把整个文件夹拷到电脑上任意位置

文件夹里有一个 启动.bat，每天双击它就行。

2 双击"启动.bat"

会自动弹出店家工作台页面，左上角显示一个二维码。

3 顾客扫码进入系统

顾客手机连店 WiFi → 打开微信扫一扫 → 扫二维码 → 开始使用。

提示：如果顾客手机微信扫不了，让顾客打开浏览器，输入店家工作台左上角显示的网址（如 192.168.1.100:3000）。

三、顾客怎么用（手机上的流程）

顾客扫码进入后，一共 5 步：

步骤做什么说明
① 登录输入昵称随便填，不用注册
② 上传文件选 PDF / Word / 图片最大 50MB
③ 预览排版翻页查看每页效果不满意可"一键转标准PDF"
④ 选择设置纸张、黑白/彩色、单双面、份数右边实时显示价格
⑤ 提交订单显示"请到前台付款"付完钱后手机显示排队状态

顾客手机上能实时看到：前面还有几个订单、大概等几分钟。

四、店家怎么用（电脑上的操作）

双击"启动.bat"后打开的页面就是店家工作台。

页面说明

位置内容用途
左上二维码顾客扫这个进系统
左下打印机状态看打印机在线还是离线
左下今日概览今天多少单、收了多少钱
右侧订单管理三个标签页（见下方）

三个标签页

① 待确认 — 顾客下单但还没付款的

  • 顾客付了钱 → 点 确认收款 → 订单自动进入打印队列
  • 也可以点 预览 看文件，或点 取消

② 打印队列 — 已确认、排队等待打印的

  • 点 开始打印 → 电脑自动打印 → 打完点 完成
  • 也可以 预览（和顾客看到的预览一样，支持翻页）
  • 不想打了可以点 取消 移出队列

③ 已完成 — 打印完的订单记录

注意：电脑重启后之前的订单记录会清空。正式版会保留。

五、店家怎么改设置

登录页面点 "店主" 快捷按钮进入后台。

改价格

"定价"标签 → 修改金额 → 点保存。改完立即生效。

规格默认单价
A4 黑白单面¥1.00
A4 黑白双面¥1.50
A4 彩色单面¥3.00
A4 彩色双面¥5.00
A3 倍率×2.00（A3=A4价×2）

管理打印机

"打印机"标签。电脑上连接的打印机会自动检测出来。也可以手动添加。

控制顾客端显示

"选项"标签。比如不想让顾客选装订方式，关掉对应开关，顾客端立刻不显示。

设置默认值

"选项"标签往下滑，设置默认纸张、颜色、份数。顾客进来就是这些默认值。

六、常见问题

Q：顾客扫不了码？

检查：① 顾客手机和电脑是不是同一个 WiFi；② 电脑 IP 可能变了，看店家工作台左上角显示的地址，让顾客手动输入。

Q：打印机没反应？

看店家工作台的打印机状态。显示"在线"就正常。显示"离线"说明打印代理停了，重新双击"启动.bat"。

Q：电脑重启后要做什么？

重新双击"启动.bat"即可。之前的订单记录会清空（当前版本限制）。

Q：顾客文件预览不出来？

一般是大文件加载慢，等一会儿。如果一直出不来，让顾客点"排版有问题 → 一键转标准PDF"重新生成。

Q：怎么同时给顾客看又自己操作？

店家工作台在电脑上，顾客用自己手机扫码访问。互不影响。

Q：想改价格/加打印机/关掉某个选项？

登录店主后台，改完保存立即生效，不用重启。

如有问题或需要升级功能，请联系系统提供方。
