# 自助打印系统 · 商家使用说明

---

## 这个系统是干什么的

顾客来店里打印文件，不用再排队等店员操作电脑。

**顾客用自己手机**扫码 → 上传文件 → 预览排版 → 选好设置 → 到前台付款 → **打印机自动打印**。

店员只需要确认收款，系统帮你算钱、排队、打印一条龙。

---

## 第一天怎么用

### 你需要准备

- 一台 Windows 电脑（Win10 或 Win11）
- 电脑已经连着打印机（平时能正常打印就行）
- 店里有 WiFi（顾客手机要连）

### 操作步骤（5 分钟）

**第一步：双击 `启动.bat`**

> 把整个文件夹拷到电脑上，双击 `启动.bat`。  
> 会弹出店家工作台页面，左上角有一个二维码。

**第二步：顾客扫码**

> 顾客手机连店 WiFi → 打开微信扫一扫 → 扫电脑屏幕上的二维码 → 进入系统。

**第三步：完事**

> 顾客在手机上操作，店家在电脑上看订单、点确认收款、自动打印。

---

## 顾客怎么用（手机上的流程）

顾客打开页面后，总共 5 步：

| 步骤 | 操作 | 说明 |
|:---|:---|:---|
| ① | 输入昵称 | 随便填，不用注册 |
| ② | 上传文件 | 支持 PDF、Word、图片 |
| ③ | 预览排版 | 滑动翻页查看每页效果，不满意可以"一键转标准 PDF" |
| ④ | 选择设置 | 纸张大小、黑白/彩色、单双面、份数、装订 |
| ⑤ | 提交订单 | 显示"请到前台付款"，付完钱等打印 |

顾客手机上能实时看到：前面还有几单、大概等几分钟。

---

## 店家怎么用（电脑上的操作）

双击 `启动.bat` 后打开的页面就是店家工作台。

### 界面说明

页面左边：

| 区域 | 用途 |
|:---|:---|
| **二维码** | 顾客扫这个进系统 |
| **打印机状态** | 看打印机是否在线 |
| **今日概览** | 今天接了多少单、收了多少钱 |

页面右边，三个标签页：

### ① 待确认

顾客下单后会出现在这里，**还没付款的**。

> 顾客付了钱，点 **"确认收款"**，订单自动进入打印队列。

按钮说明：  
**预览** — 看顾客上传的文件长什么样  
**确认收款** — 收到钱后点这个，订单进入打印队列  
**取消** — 顾客不印了，取消订单

### ② 打印队列

确认收款后的订单都在这里。

> 点 **"开始打印"** → 电脑自动把文件发给打印机 → 打完点 **"完成"**。

按钮说明：  
**预览** — 查看文件内容（和顾客看到的预览一样，支持翻页）  
**开始打印** — 发送到打印机  
**取消** — 从队列中移除此订单

### ③ 已完成

已经打印完的订单记录。

## 店家怎么改设置

登录页面点 **"店主"** 快捷按钮进入后台。

### 改价格

"定价" 标签 → 修改金额 → 点保存。改完立即可用。

> 默认价格：A4 黑白 ¥1/面，A4 彩色 ¥3/面，A3 是 A4 的 2 倍。

### 管理打印机

"打印机" 标签。系统会自动检测电脑上连接的打印机。  
也可以手动添加、修改打印机。

### 控制顾客端显示

"选项" 标签。比如不想让顾客选装订方式，关掉对应开关就行。

### 设置默认值

"选项" 标签往下滑，设置默认纸张、默认色彩、默认份数。

## 常见问题

### 顾客扫不了码？

**检查两件事：**

1. 顾客手机和电脑是不是连的**同一个 WiFi**
2. 电脑 IP 地址可能变了，看店家工作台左上角的地址，让顾客手动输入

### 打印机没反应？

看店家工作台打印机状态：

- 显示 **"在线"** → 正常，点"开始打印"试试
- 显示 **"离线"** → 打印代理没启动，重新双击 `启动.bat`

### 电脑重启后要做什么？

重新双击 `启动.bat`。  
注意：当前版本重启后之前的订单记录会清空，正式版会保留。

### 顾客文件预览不出来？

一般是文件比较大，加载需要时间。  
等一会如果还不行，点"排版有问题 → 一键转标准 PDF"。

### 只有一台电脑，怎么同时给顾客用？

店家工作台在电脑上开着，顾客用**自己的手机**扫码访问。  
互不影响。
